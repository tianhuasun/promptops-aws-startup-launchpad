# Expected Output Checklist

An AI-generated launch package must satisfy:

- [ ] Architecture present.
- [ ] IaC present.
- [ ] Terraform file plan or file contents present.
- [ ] Security baseline present.
- [ ] Cost guardrail present.
- [ ] AWS Budgets present.
- [ ] CloudWatch logs present.
- [ ] Rollback present.
- [ ] Troubleshooting present.
- [ ] No fake IDs.
- [ ] No real secrets.
- [ ] No hardcoded passwords.
- [ ] No public database by default.
- [ ] No database ingress from `0.0.0.0/0`.
- [ ] Least privilege discussed.
- [ ] Human approval gate present.
- [ ] AWS Well-Architected alignment present.
- [ ] Unknown account-specific values marked as `TODO`.
